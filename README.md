# Meditation Timer

A WeChat Mini Program for meditation timing with countdown, stopwatch, breathing guidance, audio cues, and local session history.

## Features

- Countdown and stopwatch modes
- Preset and custom durations
- Breathing guidance patterns
- Bowl and cue audio feedback
- Local meditation history and simple stats

## Project Structure

- `pages/timer/` - main timer screen
- `pages/history/` - local history screen
- `utils/audio.js` - audio playback and volume shaping
- `utils/storage.js` - local session storage helpers
- `audio/` - bundled audio assets
